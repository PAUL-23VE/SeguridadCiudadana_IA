# Diagnóstico de Zonas Peligrosas con IA

Este proyecto determina si una zona urbana es peligrosa o no, usando algoritmos de inteligencia artificial sobre mapas reales. El flujo es directo y enfocado en el diagnóstico, no en procesos internos.

## Flujo principal
1. El usuario selecciona la ciudad y la zona a analizar.
2. Ingresa datos delictivos/históricos (robos, vandalismo, etc.).
3. El sistema calcula el nivel de peligro usando lógica difusa y pesos optimizados por algoritmo genético.
4. Se aplican reglas Apriori/PRISM para explicar el diagnóstico.
5. Se pueden calcular rutas seguras o críticas usando BFS, DFS, A*, Hill Climbing.
6. El usuario recibe un diagnóstico claro: ¿la zona es peligrosa? ¿Por qué? ¿Qué factores lo determinan?

## Algoritmos principales
- Lógica difusa
- Algoritmo genético
- Apriori
- PRISM
- BFS, DFS, A*, Hill Climbing
