# 🚀 CÓMO EJECUTAR EL PROYECTO

## Instalación de Dependencias
```cmd
pip install -r requirements.txt
```

## Ejecución del Sistema
```cmd
python main.py
```

## Selección de Modo
1. **Análisis Completo** (Opción 1): Analiza todas las zonas de Ambato
2. **Análisis Específico** (Opción 2): Selector interactivo de zona circular

## Resultados
- **Mapas HTML**: Se guardan en la carpeta `output/`
- **Gráficas**: Permanecen abiertas para presentaciones
- **Diagnóstico**: Mostrado en consola con niveles de riesgo

## Estructura de Archivos Importantes
- `main.py` - Programa principal
- `src/algoritmos/` - 7 algoritmos de IA
- `src/core/` - Motor de diagnóstico
- `requirements.txt` - Dependencias necesarias

**¡El sistema está configurado específicamente para Ambato, Ecuador!**
